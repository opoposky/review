## git 常用指令

```js
    git clone [project] // 克隆分枝
    git pull // 拉取远程分支
    git status // 检查文件状态
    git add [filePath] // 添加文件状态
    git commit -m/am [message] // 暂存提交 直接提交  

    git branch [-a] // 查看当前 -a 所有分支
    git branch [name] // 创建本地分支

    git checkout [name] // 切换分支
    git checkout [filepath] // 还原路径文件
    git checkout -b [name] // 创建新分支并切换

    git push  // 推送远程分支
    git push origin [name] // 推送分支到远程分支上

    

```
